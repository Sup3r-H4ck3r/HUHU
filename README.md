# Validate Data

## Cài đặt môi trường

```shell
python -m venv my_env
```
## Kích hoạt môi trường

Nếu chạy trên windows
```shell
my_env\Scripts\activate
```
Nếu chạy trên linux/macOS
```shell
source my_env/bin/activate
```
## Cài đặt các thư viện cần thiết

```shell
pip install -r requirements.txt
```
## Run

```shell
python main.py
```
